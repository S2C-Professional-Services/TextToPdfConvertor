To run this plugin and convert text files in pdf files
follow the steps:

1) extract the contained .rar file
2) open TextToPDFPlugin.exe.config file to change below things:
	-> DirectoryPath value - input folder path contains .txt files (here it is "D:\TextFiles")
	-> OutputDirectoryPath - output folder path where you want to save the generate pdf files (here is "D:\PdfFiles")
3) save and close the .config file and then run the TextToPDFPlugin.exe

NOTE: make sure current logged in user has rights to creatr directory and files in the folder specified.
